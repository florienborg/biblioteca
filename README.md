## Instalación
Este proyecto se ha desarrollado en la version `v18.12.0` o superior de node.

Para instalar las dependencias:

  `npm install`

Para ejecutar la App en modo desarrollo:

  `npm start`

Para ejecutar los tests unitarios: 

  `npm run test`
